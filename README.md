# gemini-canvas-mission

## 앱 이름: 알람 타이머
## 카테고리: 유틸리티 앱

### 배포 링크
https://gemini.google.com/share/6324a015019d

### 이 앱을 만든 이유
- 아침에 분주하게 준비할 때 일일이 핸드폰으로 시간을 확인해야 하는 번거로움을 해결하고자 만들었습니다.
- 집에서 준비할 때 목표 시간을 설정해 목표 시간 50분 전부터 10분 간격으로 남은 시간을 제공받고자 합니다.
- 그 외에도 무언가에 집중하다가 약속 시간을 고려하지 못 하는 것을 예방하기 위해 사용하면 좋을 것 같습니다.

### 주요 기능
- 목표 시간을 지정할 수 있습니다.
- 지정 시간 50분 전부터 10분 간격으로 "N분 남았습니다."라고 소리로 알려줍니다.

## 앱 이름: 라이어 게임
## 카테고리: 게임

### 배포 링크
https://gemini.google.com/share/ea055a5e1769

### 이 앱을 만든 이유
- 라이어 게임은 최소 3명이 모여야 할 수 있는 게임입니다.
- 그래서 혼자서도 라이어 게임을 할 수 있도록 AI와 함께하는 라이어 게임을 만들었습니다.

### 주요 기능
- 게임에 참여할 인원 수를 결정할 수 있습니다.
- 라이어 또는 시민이 랜덤으로 배정됩니다.
- 자신의 차례에 특정 키워드에 대해 설명합니다.
- 모든 인원의 설명이 끝난 후에는 투표를 진행합니다.

### AI 기능 (해당되는 경우)
- 게임에 참여한 AI들이 각각의 설명들을 분석하여 누가 라이어일지 판단하도록 했습니다.
- 이를 통해 조금 더 능동적인 참여자가 될 수 있도록 했습니다.
